Kamon

Kamon is a set of tools that will help you monitor your reactive applications.

Akka actors metrics and tracing

kamon offers a wide set of metrics related to Akka Toolkit, such as:

  • Messages rates
  • Actor mailbox queue size
  • Thread pools health
  • Actor hierarchies
  • Message Tracing
